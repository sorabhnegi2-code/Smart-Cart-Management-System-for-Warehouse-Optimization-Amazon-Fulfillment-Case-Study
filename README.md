# Smart-Cart-Management-System-for-Warehouse-Optimization-Amazon-Fulfillment-Case-Study
Designed a smart warehouse cart management system using UML, improving cart tracking, assignment, and operational efficiency.
## 🧩 Key Features

- Real-time cart tracking system  
- Automated cart assignment  
- Performance monitoring dashboard  
- Reduced manual communication  
- Improved workflow efficiency  

---

## 🛠️ System Design

- Use Case Diagrams  
- Entity Relationship Diagram (ERD)  
- Data Flow Diagrams (DFD)  
- Class Diagram  
- 3-Tier Architecture  

---

## ⚠️ Problem Solved

- Cart unavailability delays  
- Lack of real-time tracking  
- Inefficient communication  

---

## 💡 Proposed Solution

- GPS / tracking-based cart monitoring  
- Intelligent assignment system  
- Centralized dashboard for managers
